# Issue 3: [Phase 3-1] ホーム画面 (Home Screen) & 新規イベント追加機能の実装

## 概要
お出かけ予定の一覧をカード形式で表示し、フローティングアクションボタン (FAB) から新規イベントを追加・削除できるホーム画面を実装します。

## タスクリスト
- [ ] ホーム画面 (`app/index.tsx`) の基本レイアウト作成
- [ ] イベントカードコンポーネントの作成
  - イベント名、日付の表示
  - 準備状況（持ち物のチェック率）を表す進捗インジケータ（プログレスバー等）の表示
- [ ] 「新規イベント追加」用モーダル（またはボトムシート）の実装
  - イベント名、日付（DatePicker等、またはテキスト入力）の入力フォーム
  - バリデーション（空文字入力防止など）
- [ ] イベントの削除機能の実装（長押し、またはスワイプによるコンテキストメニュー/ゴミ箱アイコン）

## 検証項目
- [ ] 新規イベントがモーダルから正常に追加され、ホーム画面の一覧に即時反映されること
- [ ] 持ち物のチェック状態に応じて、カード上の進捗バーが正しく動的に変化すること
- [ ] イベントの削除操作時に、確認アラートを挟んで正常に削除できること
