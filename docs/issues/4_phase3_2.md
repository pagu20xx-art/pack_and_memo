# Issue 4: [Phase 3-2] 準備画面 (Checklist Screen) & 持ち物チェックリスト機能の実装

## 概要
選択したお出かけイベントに紐づく持ち物リストを表示し、チェックボックスのトグルや、新規持ち物の追加を行える準備画面（チェックリスト画面）を実装します。

## タスクリスト
- [ ] 準備画面 (`app/checklist/[id].tsx`) の基本UI・ルーティングの実装
  - イベント名と予定日をヘッダーに分かりやすく表示
- [ ] 持ち物リストのレンダリング
  - チェックボックス、持ち物名、削除用のゴミ箱アイコン
- [ ] チェックボックスのタップ時に `useEvents` フックを介して SQLite の状態を即時更新
- [ ] 画面下部、またはリスト末尾に「持ち物を追加」するためのシンプルな入力フォームを実装
- [ ] 「思い出を記録する」ボタンを配置し、思い出画面へ遷移できる導線を作成

## 検証項目
- [ ] ホームからカードタップで正しいイベントIDの準備画面へ遷移し、そのイベントに紐づくアイテムのみが表示されること
- [ ] 持ち物のチェックをトグルした際、チェック状態が保存され、ホームに戻った際も進捗率が連動していること
- [ ] 新規持ち物をフォームから追加した際、リストの末尾にすぐ追加されること
- [ ] 個別の持ち物を削除できること
